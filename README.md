# Landing Page Project

A responsive landing page built from scratch based on a provided design mockup - part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## About This Project

This project involves recreating a full web page layout from a design reference using only HTML and CSS. No frameworks, no shortcuts - just clean markup and handwritten styles.

The goal is to translate a static design into a working, structured web page while applying the core concepts covered in the curriculum so far.

## Skills Demonstrated

- Semantic **HTML5** structure
- **CSS Flexbox** for layout and alignment
- Responsive design fundamentals
- Working from a design spec / mockup
- Organizing and writing clean, readable code

## What I Learned

Building this page pushed me to think about layout from the ground up. Flexbox made it possible to control alignment and spacing without hacks - once it clicked, it changed how I approach every layout problem. The biggest takeaway: breaking a page into rows and columns mentally before writing a single line of CSS saves a lot of frustration.

## Tech Stack

- HTML5
- CSS3 (Flexbox)
